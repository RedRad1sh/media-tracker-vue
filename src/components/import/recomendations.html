Год
Как настроить систему рекомендаций? Выберите списки, на основе которых будут формироваться рекомендации (фильмы, игр или книги). Для этого в настройках выберите нужные списки, которые хотите использовать для рекомендаций.
Выберите тип контента, который вы хотите, чтобы система рекомендовала: фильмы, игры, книги. Для этого просто выберите нужный тип в настройках.
Отфильтруйте рекомендации по жанрам и году выпуска.
Нажмите кнопку “Сформировать рекомендации”
Либо нажмите кнопку “Удиви меня”, чтобы получить случайную рекомендацию фильма, книги или игры по вашим предпочтениям.
